 Number Guessing Challenge (Java Console Game)

This interactive Java console game tests players' intuition by guessing a hidden number within a set range and attempt limit. The application supports extended play sessions with performance tracking.

Core Functionality
Adjustable Difficulty: Random target generated from 1-100 (modifiable through code constants)
Attempt Restriction: Maximum 7 tries allowed per game round
Extended Sessions: Complete 3 full rounds with session-wide statistics
Smart Feedback: Real-time clues reveal if guess is too high or too low
Performance Rewards: Earn 10 base points + 5 bonus per unused attempt
Crash Protection: try-catch with InputMismatchException safeguards invalid entries
Session Report: Final tally shows rounds completed, wins achieved, cumulative score
